Behind the Scenes

The Application

GMart is a storefront application developed in Python, from scratch, using the Django framework. It includes an implementation of the Django admin backend for managing products, categories and users. Additionally, a custom implementation of the authentication framework allows for front-end user account management.

The storefront allows members to shop for products, storing selections in their shopping cart. Members can view the contents of their carts, remove items, adjust quantities, etc.

The application uses bootstrap for CSS styling throughout the site, including the carousel and card components for product presentation.

And finally, vanilla javascript is used to create dynamic increase/decrease buttons to modify product quantities in the DOM, which are passed back to Django, creating dynamic SPA-type functionality for an improved user experience.

The Deployment

GMart has been deployed to a cloud environment, on the AWS platform. The project is maintained with version control, using GIT, and pushed to a Github repository. AWS deployment is configured with a CodePipeline to automatically recognize and deploy changes from the Github repository.
